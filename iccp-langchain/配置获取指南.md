# 配置获取指南

## 环境变量配置

项目需要以下环境变量才能正常运行。请在 `.env` 文件中配置：

### 必需配置

```bash
# OpenAI API 配置
OPENAI_API_KEY=your_openai_api_key_here
OPENAI_BASE_URL=https://api.openai.com/v1  # 或使用代理地址

# 数据库配置（可选，默认使用 SQLite）
DATABASE_URL=sqlite:///./iccp.db

# 架构配置（可选，默认为 true）
USE_NEW_ARCHITECTURE=true
```

### 可选配置

```bash
# Supabase 配置（如果使用 Supabase）
SUPABASE_URL=your_supabase_url
SUPABASE_KEY=your_supabase_key

# 微信小程序配置（如果使用微信登录）
WECHAT_APP_ID=your_wechat_app_id
WECHAT_APP_SECRET=your_wechat_app_secret

# 其他 LLM 配置
ANTHROPIC_API_KEY=your_anthropic_key  # 如果使用 Claude
ZHIPU_API_KEY=your_zhipu_key  # 如果使用智谱 AI
```

## 如何获取 API Key

### OpenAI API Key

1. 访问 [OpenAI Platform](https://platform.openai.com/)
2. 注册/登录账号
3. 进入 API Keys 页面
4. 点击 "Create new secret key"
5. 复制生成的 key 到 `.env` 文件

### 使用代理服务

如果无法直接访问 OpenAI，可以使用代理服务：

```bash
# 示例：使用国内代理
OPENAI_BASE_URL=https://api.openai-proxy.com/v1
OPENAI_API_KEY=your_proxy_api_key
```

常见代理服务：
- OpenAI-SB
- API2D
- CloseAI
- 等等

## 快速开始

1. 复制示例配置文件：
```bash
copy .env.example .env
```

2. 编辑 `.env` 文件，填入你的配置

3. 启动项目：
```bash
# 激活 conda 环境
conda activate opc

# 启动服务
python -m app.main
```

4. 访问 API 文档：
   - Swagger UI: http://localhost:8000/docs
   - ReDoc: http://localhost:8000/redoc

## 常见问题

### Q: 没有 OpenAI API Key 怎么办？

A: 可以使用其他 LLM 提供商：
- 智谱 AI (GLM-4)
- 阿里云通义千问
- 百度文心一言
- 等等

需要修改 `app/llm/client.py` 中的配置。

### Q: 如何测试配置是否正确？

A: 启动服务后访问健康检查端点：
```bash
curl http://localhost:8000/health
```

### Q: 数据库配置

A: 默认使用 SQLite，无需额外配置。如需使用 PostgreSQL：

```bash
DATABASE_URL=postgresql://user:password@localhost:5432/dbname
```
